# Localization Guide for lora-scripts-anima

## i18n Architecture

The frontend uses a custom i18n module (`frontend/js/i18n.js`) with JSON translation files.

### Files

| 文件 | 用途 |
|------|------|
| `frontend/i18n/zh-CN.json` | 简体中文 |
| `frontend/i18n/en-US.json` | English |
| `frontend/js/i18n.js` | i18n 运行时模块 |

### Section Structure (13 top-level sections, 676 keys)

```
nav          — 导航菜单
monitor      — 训练监控页
section      — 表单分区标题
field        — 表单字段标签/提示
common       — 通用按钮/状态
preset       — 预设管理
tagger       — Tagger 打标器
tagEditor    — 标签编辑器
tools        — 工具页
settings     — UI 设置
about        — 关于页
environment  — 环境管理页
opt          — 附加选项
```

## Mandatory Rules

1. **所有用户可见文本必须走 i18n**，禁止在 JS/HTML/CSS 中硬编码中文或英文字符串
2. **新增 key 必须同时加到 zh-CN.json 和 en-US.json**，且 key 命名一致
3. **JSON 语法严格**：不允许尾随逗号，key 必须双引号
4. **修改 JSON 后必须验证**：`python -c "import json; json.load(open('frontend/i18n/zh-CN.json')); json.load(open('frontend/i18n/en-US.json'))"`

## Usage in JS

```js
// 在 Alpine.js 组件中
this.t('section.key')           // 返回翻译文本
this.t('section.key', 'fallback') // key 不存在时返回 fallback
```

## Common Mistakes

- ❌ 硬编码中文：`` `你好，${name}` `` → ✅ `this.t('greeting.hello') + name`
- ❌ 只加了一个语言文件
- ❌ JSON 尾随逗号：`"key": "value",` 在最后一个元素后
- ❌ key 命名不一致（如 zh-CN 用 `myKey` 但 en-US 用 `my_key`）
- ❌ 日志过滤按钮（All/Error/Warn/Info）硬编码英文 → 必须用 `t('monitor.logLevelAll')` 等
- ❌ 图表占位标签硬编码英文 → 必须用 `t('monitor.chartLossAverage')` 等
- ❌ 优化器默认值在多处重复硬编码 → 统一在 `constants.js` 的 `OPTIMIZER_DEFAULTS` 中定义

## Verification Commands

```bash
# 验证 JSON 语法
python -c "import json; json.load(open('frontend/i18n/zh-CN.json', encoding='utf-8')); json.load(open('frontend/i18n/en-US.json', encoding='utf-8'))"

# 检查 key 一致性
python -c "
import json
zh = json.load(open('frontend/i18n/zh-CN.json', encoding='utf-8'))
en = json.load(open('frontend/i18n/en-US.json', encoding='utf-8'))
def keys(d, p=''): return {f'{p}.{k}' if p else k for k,v in d.items()} | \
    {sk for k,v in d.items() if isinstance(v,dict) for sk in keys(v, f'{p}.{k}' if p else k)}
only_zh = keys(zh)-keys(en); only_en = keys(en)-keys(zh)
print('Only zh:', only_zh or 'none'); print('Only en:', only_en or 'none')
"
```
