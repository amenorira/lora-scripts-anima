Parameter Reference

Complete reference for all LoRA training parameters. Hover over any field in the training form to see its description.

Model

pretrained_model_name_or_path — Base model path. For SD/SDXL provide a .safetensors file; for DiT models (Anima) provide the model directory.

vae — Optional VAE model path. Anima requires QwenImage VAE.

resume — Path to a save_state checkpoint to resume interrupted training.

model_train_type — Training type: SD LoRA, SDXL LoRA, Anima LoRA, Flux LoRA, or SD3 LoRA.

Dataset

train_data_dir — Directory containing training images in subfolders, each with .txt caption files.

resolution — Training resolution as width,height. Must be a multiple of 64 for SD/SDXL, 16 for Anima.

enable_bucket — Enable ARB (Aspect Ratio Bucketing) for mixed aspect ratio datasets.

Training

max_train_epochs — Maximum number of training epochs. One epoch = one pass through the entire dataset.

train_batch_size — Number of images processed simultaneously. Higher values require more VRAM.

gradient_accumulation_steps — Accumulate gradients over multiple steps. Effectively increases batch size without VRAM increase.

Learning Rate

learning_rate — Overall learning rate. Typical values: 1e-4 for SD, 5e-5 for SDXL, 1e-4 for Flux.

unet_lr — U-Net / backbone learning rate. Set separately from text encoder for fine control.

lr_scheduler — Learning rate schedule. cosine_with_restarts is a good default for most cases.

optimizer_type — AdamW8bit offers a good balance of speed and memory. Prodigy is adaptive.

Network

network_dim — LoRA rank / network dimension. Common range: 4–128. Lower values use less VRAM but may reduce quality.

network_alpha — Scaling factor. Typically equal to or half of network_dim. Lower alpha needs higher learning rate.

Speed & VRAM

mixed_precision — bf16 recommended for RTX 30+ GPUs. fp16 for older cards. no for debugging.

xformers — Enable xformers memory-efficient attention. Recommended for most GPUs.

cache_latents — Pre-compute VAE latents to disk, saving VRAM during training.

For detailed parameter explanations, hover over any field label in the training form.
