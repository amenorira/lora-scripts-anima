Parameters

Parameter reference for LoRA training

Refer to the schema documentation for full parameter explanations. Key parameters:

pretrained_model_name_or_path: Base model file path (.safetensors for SD/SDXL; directory for DiT models).

train_data_dir: Training dataset directory containing image subfolders with caption .txt files.

resolution: Training image resolution (width,height). Must be a multiple of 64 for SD/SDXL, 16 for Anima.

max_train_epochs: Maximum number of training epochs.

train_batch_size: Number of images processed simultaneously. Higher values use more VRAM.

learning_rate: Overall learning rate. Common values: 1e-4 for SD, 5e-5 for SDXL, 1e-4 for Flux.

network_dim: LoRA rank / network dimension. Common range: 4-128. Lower values use less VRAM.

network_alpha: Scaling factor. Typically equal to or half of network_dim.

optimizer_type: Optimizer selection. AdamW8bit is a good default for most use cases.

mixed_precision: Training precision. bf16 recommended for RTX 30+ series.
