# 标签编辑器三栏布局重设计

> 状态: 设计已批准 · 日期: 2026-06-04

## 1. 问题陈述

当前标签编辑器的核心痛点：

1. **顶部工具栏堆叠过高**：搜索栏 + 复制粘贴栏 + 多选栏 + 批量操作栏（4 行）严重压缩图片网格可视区域，1080p 屏幕下尤为明显
2. **编辑流程绕**：编辑单张图片标签需要点击卡片 → 等待详情抽屉滑出 → 在抽屉内编辑 → 关闭抽屉 → 再点下一张，流程冗长
3. **批量/多选 UI 功能重叠**：多选浮动栏和批量操作栏均可添加/删除标签，用户困惑该用哪个
4. **卡片内联编辑体验差**：空间小、容易误触（当前已隐藏）

## 2. 布局架构

```
┌─────────────────────────────────────────────────────────┐
│  顶栏: 搜索 | 快速筛选 | 排序 | 选择控制 | 保存 | 面板折叠      │  ← 40px
├────────────┬──────────────────────────────┬─────────────┤
│  标签云     │                              │  编辑面板     │
│  · 目录栏   │                              │  · 预览图    │
│  · 撤销/重做 │      图片网格 (auto-fill)     │  · 标签芯片   │
│  · 标签搜索  │                              │  · 添加标签   │
│  · AND/OR  │                              │  · 批量操作   │
│  · 频率列表  │                              │  · 标签统计   │
│  200px     │      flex-grow               │  320px      │
├────────────┴──────────────────────────────┴─────────────┤
│  状态栏: 总数 | 筛选 | 已选 | 已修改 | 分页 | 保存状态        │  ← 32px
└─────────────────────────────────────────────────────────┘
```

### 响应式行为

| 屏幕宽度 | 行为 |
|---------|------|
| > 1600px | 三栏全展开，左 200px + 右 320px |
| 1200-1600px | 左栏折叠为 48px 图标条（hover 展开），右栏缩至 260px |
| < 1200px | 右栏变为浮层覆盖，左栏图标模式 |

## 3. 核心组件

### 3.1 顶栏（40px，单行）

- 搜索输入框 + 正则开关
- 快速筛选胶囊：全部 | 无标签 | 已修改，显示各数量
- 排序下拉：名称 / 标签数 / 修改时间
- 选择控制：全选 / 反选 / 取消全选
- 保存按钮：有修改时高亮 + 显示待保存数
- 左右面板折叠按钮

### 3.2 左栏 —— 标签云面板（200px，可折叠至 48px）

- 目录面包屑 + 重新加载按钮
- 撤销/重做按钮 + 历史栈计数
- 标签搜索（过滤标签云）
- AND/OR 筛选逻辑切换
- 当前激活的筛选条件（含/排除标签，可点击移除）
- 频率列表：频率条 + 标签名 + 数量；左键=加入筛选，右键=排除
- 可折叠为 48px 图标条，hover 展开临时预览

### 3.3 中栏 —— 图片网格

CSS Grid `repeat(auto-fill, minmax(180px, 1fr))`，每张卡片：

- 缩略图（aspect-ratio: 1）
- 左上：选择复选框
- 右上：已修改标记 "M"
- Hover 时：底部浮出前 3 个标签胶囊 + 文件名

交互：
| 操作 | 效果 |
|------|------|
| 单击 | 选中 + 右侧显示标签 |
| Shift+单击 | 范围多选 |
| 双击 | 编辑聚焦（右侧焦点） |
| 拖拽矩形 | 框选 |
| Hover | 显示前 3 个标签 + 文件名 |

### 3.4 右栏 —— 编辑面板（320px）

根据选中数量自动切换模式：

**模式 A：单图（选中 1 张）**
- 缩略小图 + 文件名
- 上一张/下一张导航
- 标签芯片：拖拽排序、点击 × 删除、双击编辑标签名
- 添加标签输入框（自动补全）
- 排序按钮
- 芯片/文本视图切换
- 复制/粘贴标签按钮

**模式 B：批量（选中 ≥2 张）**
- 已选图片计数
- 批量添加（输入 + 前/后位置）
- 批量移除（输入 + 自动补全）
- 批量替换（旧标签 → 新标签）
- 去重 / 排序
- 选中图片的标签统计（频率分布）
- 确认按钮（操作前二次确认）

**无选中时**：占位文案

### 3.5 底部状态栏（32px）

总数 | 筛选显示 | 已选 | 已修改 | 分页控件 | 每页数量选择 | 未保存指示器

## 4. 交互流程

### 即时反馈
- 标签修改不自动保存，立即显示"M"标记 + 芯片变色
- 状态栏"已修改"计数实时更新
- 标签云频率增量更新（无需重新请求后端）

### 面板联动
```
单击1张 → 右侧切「单图编辑」 → 显示该图标签
多选N张 → 右侧切「批量编辑」 → 显示标签统计 + 批量操作
左栏筛选 → 中间过滤 → 选中项如在筛选外则自动取消 → 右侧恢复占位
Esc     → 取消所有选中 → 右侧恢复占位
```

### 键盘快捷键

| 按键 | 作用 |
|------|------|
| Ctrl+S | 保存所有修改 |
| Ctrl+Z | 撤销 |
| Ctrl+Shift+Z | 重做 |
| Ctrl+A | 全选当前筛选结果 |
| Ctrl+C | 复制选中图片标签 |
| Ctrl+V | 粘贴标签到选中图片 |
| ← → | 单图模式导航 |
| Esc | 取消所有选中 |
| Ctrl+F | 聚焦搜索框 |

### 防误操作
- 保存前确认对话框（显示修改数量）
- 浏览器关闭/刷新时 beforeunload 拦截
- 批量删除标签二次确认
- 撤销栈上限 200

### 面板折叠
- 左栏：点击折叠 → 48px 图标条，hover 展开预览，离开收回
- 右栏：无选中自动折叠，选中自动展开，可手动固定
- 三栏全折叠 → 图片网格满屏（适合纯浏览）

## 5. 后端 Bug 修复

### 必须修复

**B1 [严重] `/save` 路径遍历漏洞** — `backend/tageditor/routes.py:166`
- 问题：单图保存端点未校验路径，可写入任意位置
- 修复：与 `/save-all` 一致，校验 `cap_resolved.parent == img_resolved.parent`

### 建议修复

**P1 `scan_images` 性能** — `backend/tageditor/core.py:71`
- 问题：对 6 种扩展名各调用一次 `rglob`，扫描 6 遍
- 修复：单次 `rglob("*")` + 扩展名过滤

## 6. 技术约束

- **框架**：保持 Alpine.js（与项目其他路由一致）
- **CSS**：保持 CSS 自定义属性体系，不引入新框架
- **后端 API**：保持现有接口不变（13 个端点），仅修 Bug
- **i18n**：所有用户可见文本使用 `t()` 函数，key 命名遵循 `tagEditor.*` 规范
- **禁止修改**：`vendor/sd-scripts/`
